Como Usar o Sistema

Dashboard

O dashboard mostra um resumo geral do seu sistema, incluindo:

  • Total em cheques
  • Cheques vencendo hoje
  • Próximos vencimentos
  • Total de empresas cadastradas

Gerenciando Cheques

Para gerenciar seus cheques:

  • Clique em "Novo Cheque" para adicionar um novo
  • Use a lista de cheques para visualizar todos os registros
  • Utilize os filtros para encontrar cheques específicos
  • Edite ou exclua cheques conforme necessário

Empresas de Fomento

Gerencie suas empresas de fomento:

  • Adicione novas empresas
  • Edite informações existentes
  • Visualize todos os cheques por empresa

Notificações

Configure suas notificações para:

  • Cheques vencendo hoje
  • Próximos vencimentos
  • Atualizações importantes

Perguntas Frequentes

Como faço backup dos meus dados?

Acesse o menu de configurações e clique em "Backup". Você pode exportar seus dados para um arquivo JSON.

Como restauro meus dados?

No menu de configurações, use a opção "Restaurar" e selecione seu arquivo de backup.

Como altero minha senha?

Acesse seu perfil através do menu de configurações e clique em "Alterar Senha".

Como filtro meus cheques?

Na lista de cheques, use o botão de filtro para acessar todas as opções de filtragem disponíveis.
