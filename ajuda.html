Gerenciamento de Empresas

Como cadastrar uma nova empresa?

  1. Acesse a página "Empresas" no menu lateral
  2. Clique no botão "Nova Empresa"
  3. Preencha os dados da empresa (CNPJ, Nome, etc.)
  4. Clique em "Salvar"

Como editar uma empresa?

  1. Localize a empresa na lista
  2. Clique no ícone de edição (lápis)
  3. Atualize os dados necessários
  4. Clique em "Salvar"

Controle de Cheques

Como registrar um novo cheque?

  1. Selecione a empresa na página inicial
  2. Clique em "Novo Cheque"
  3. Preencha os dados do cheque (número, valor, vencimento)
  4. Selecione o status do cheque
  5. Clique em "Salvar"

Como atualizar o status de um cheque?

  1. Localize o cheque na lista
  2. Clique no status atual
  3. Selecione o novo status
  4. Confirme a alteração

Relatórios e Análises

Como gerar um relatório?

  1. Acesse a página "Relatórios"
  2. Selecione o tipo de relatório
  3. Defina o período desejado
  4. Aplique filtros adicionais (opcional)
  5. Clique em "Gerar Relatório"

Como exportar dados?

  1. Gere o relatório desejado
  2. Clique no botão "Exportar"
  3. Selecione o formato (Excel ou PDF)
  4. Aguarde o download

Configurações do Sistema

Como configurar notificações?

  1. Acesse a página "Configurações"
  2. Vá até a seção "Notificações"
  3. Ative/desative as opções desejadas
  4. Configure os dias de antecedência para alertas
  5. Clique em "Salvar"

Como fazer backup dos dados?

  1. Acesse "Configurações"
  2. Vá até a seção "Backup"
  3. Configure a frequência do backup automático
  4. Ou clique em "Fazer Backup Agora"

Perguntas Frequentes

Como recuperar minha senha?

Clique em "Esqueci minha senha" na tela de login e siga as instruções enviadas para seu e-mail.

Como alterar dados da minha conta?

Acesse "Configurações" e vá até a seção "Perfil" para atualizar seus dados.

Como filtrar cheques por status?

Use os filtros disponíveis no topo da lista de cheques para selecionar o status desejado.

O sistema funciona offline?

Não, o sistema requer conexão com a internet para funcionar corretamente.
